# Dictionaries

- Converting a list of tuples into a Dictionary

	```
	In [1]: name_and_ages = [('Alice',32),('Bob',48),('Charlie',28),('Daniel',33)]

	In [2]: d = dict(name_and_ages)
	
	In [3]: d
	Out[3]: {'Alice': 32, 'Bob': 48, 'Charlie': 28, 'Daniel': 33}
	
	```
