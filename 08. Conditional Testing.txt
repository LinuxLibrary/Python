Conditional Testing :
---------------------

>>> min, max = 8,9
>>> print min
8
>>> Print max
9
>>> if min < max:
... 	print min, "is less than", max
...		<RETURN>
8 is less than 9
>>> if min > max:
...	 print min, "is greater than", max
...		<RETURN>
>>> if min > max:
...	 print min, "is equal to", max
...		<RETURN>

>>> filecount = 0

>>> filecount = 1
